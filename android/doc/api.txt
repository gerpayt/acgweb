登陆
地址 /api/login
参数 username 账号
     password 密码（明文）
返回 登录失败 {"error": "101", "message": "..."}
     登陆成功 {"access_token": "...", "name": "..", "uid": ".."}
         access_token 访问令牌
         name 名字
         uid 学号

注销
地址 /api/logout
参数 token 令牌
返回 {'success': True, 'message': '...'}

注册
地址 /api/register
参数 username 学号
     password 密码
     name 姓名
     email 电子邮箱
     mobile 手机号
     reqcode 注册码
返回 注册成功 {'success': True, 'message': '...'}
     注册失败 {'error': '120', 'message': ['...', '...', ...]}

活动列表
地址 /api/activitylist
参数 token 令牌
     pagenum 页数（非必填）
返回 [{"id":.., "title": ..., ... }, {...}, ...]
        id 活动唯一标识
        title 活动标题
        start_time 活动开始时间
        venue 活动场地
        status 活动状态
        type 活动类型
        duties 任务数组
            uid 用户学号
            name 用户姓名
            status 任务状态

活动详细
地址 /api/activitydetail
参数 token 令牌
     activity_id 活动ID
返回 {"id": ..., "title": ..., ...}
        id 活动唯一标识
        title 活动标题
        work_start_time 值班开始时间
        start_time 活动开始时间
        end_time 活动结束时间
        remark 活动备注
        venue 活动场地
        status 活动状态
        type 活动类型
        duties 任务数组
            id 任务ID
            uid 用户学号
            name 用户姓名
            mobile 手机号
            mobile_type 手机号类型
            mobile_short 手机短号
            status 任务状态
            operations 可操作的数组
                name 操作名称
                title 操作显示名称
                color 显示颜色
                content 提示框信息
                require 是否需要填写附加信息
